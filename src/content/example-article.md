## Welcome

This is a sample article demonstrating the content pipeline. Articles are written in Markdown with frontmatter metadata.

## How to Add Articles

1. Create a new `.md` file in `src/content/`
2. Add frontmatter with `title`, `description`, and `date`
3. Write your content in Markdown below the frontmatter
4. The article will automatically appear on the homepage and get its own SEO-optimized page

## SEO Features

Every article automatically gets:

- Unique meta title and description
- Open Graph tags for social sharing
- Clean URL structure (`/your-article-slug`)
- Sitemap inclusion
- Static HTML generation for fast load times
